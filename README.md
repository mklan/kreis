# kreis

Game with the goal to draw a perfect circle.

![Gameplay as animated Gif](https://github.com/mklan/kreis/blob/master/demo.gif)

[Demo](https://mklan.github.io/kreis/)

## develop

`yarn start`

## deploy to gh-pages

`yarn deploy`
